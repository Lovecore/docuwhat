## Code Walkthroughs

### Component Development

Learn how to create custom MDX components:

```javascript
// Custom video component with chapters
export function VideoWithChapters({ src, chapters }) {
  return (
    <div className="video-container">
      <video src={src} controls />
      <div className="chapters">
        {chapters.map((chapter, index) => (
          <button key={index} onClick={() => seekTo(chapter.time)}>
            {chapter.title}
          </button>
        ))}
      </div>
    </div>
  );
}
```

## Best Practices for Video Content

### Video Format Guidelines

- **Resolution**: 1920x1080 (Full HD) recommended
- **Format**: MP4 with H.264 codec for best compatibility
- **File Size**: Optimize for web delivery (< 100MB per video)
- **Audio**: Clear narration with minimal background noise

### Embedding Videos

You can embed videos in multiple ways:

1. **Local Videos**: Store in `/public/videos`
2. **External Videos**: Link to YouTube, Vimeo, etc.
3. **Streaming**: Use CDN for better performance

<Callout type="warning" title="Performance Tip">
  For large video files, consider using a CDN or video hosting service to improve loading times.
</Callout>
